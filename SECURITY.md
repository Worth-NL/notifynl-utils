## Reporting a Vulnerability

If you discover a security vulnerability in this project, please follow the steps below to report it:

1. **Do not open an issue** on GitHub for security vulnerabilities. Instead, send an email to [info@worth.nl](mailto:info@worth.nl) with the details of the vulnerability.
2. Include the following information in your report:
   - A detailed description of the vulnerability.
   - Steps to reproduce the issue.
   - Any potential impact or risk associated with the vulnerability.
   - Your contact information (optional).

3. Once the vulnerability is confirmed, we will work on a fix and release a security update as soon as possible.

## Security Updates

Security updates will be released as new versions of the project. We recommend that you always use the latest version to ensure you have the most up-to-date security fixes.

## Responsible Disclosure

We are committed to responsible disclosure and will work with you to ensure that vulnerabilities are addressed promptly and transparently.

## Contact

For any security-related questions or concerns, please contact our security team at [info@worth.nl](mailto:info@worth.nl).
